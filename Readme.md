# Superstore Retail Analytics Dashboard

## Project Overview
End-to-end retail sales analytics project analyzing 9,994 transactions 
from 2014-2017 using Excel, MySQL and Power BI.

## Tools Used
- Excel — Data cleaning and transformation
- Python — Data import automation
- MySQL — Database design and SQL analysis
- Power BI — Interactive dashboard

## Project Structure
- superstore_analysis_queries.sql — 15 SQL queries + 23 views
- import_data.py — Python MySQL import script
- fix_dates.py — Date fixing script
- screenshots/ — Dashboard screenshots

## Key Insights
- Revenue grew 20.4% YoY in 2017
- 100% of high discount orders are loss-making
- Top 20% customers generate 80% of revenue (Pareto Rule)
- Furniture category loses money despite high sales volume
- South region is 32% below national average

## Dashboard Pages
1. Executive Summary — KPI cards, revenue trends, geographic map
2. Customer Analysis — RFM segmentation, Pareto analysis, retention
3. Operations Analysis — Discount impact, shipping efficiency, heatmap

## Skills Demonstrated
SQL: Window Functions, CTEs, Subqueries, UNION ALL, Views
Excel: Data cleaning, derived columns, date transformation
Power BI: DAX measures, interactive visuals, drill-through
Python: Pandas, mysql-connector, data transformation
